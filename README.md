# 🚀 Aditya Singh Portfolio

A next-generation interactive software engineering portfolio built to showcase my journey from Data Structures & Algorithms to scalable backend systems, distributed architectures, and production-grade software engineering.

## 🎯 Overview

This portfolio is designed as an Engineering Control Deck rather than a traditional personal website.

It visualizes:

* Knowledge Evolution Journey
* Data Structures & Algorithms Roadmap
* Backend Engineering Expertise
* System Design Concepts
* Architecture Thinking
* Project Showcase
* Interactive Engineering Visualizations

The goal is to present software engineering growth as an evolving system rather than a static resume.
